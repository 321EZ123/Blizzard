Available API Endpoints

Main Proxy:
GET /api/index.js?url=<encoded_url>
Google Proxy:
GET /api/google/index.js?url=<encoded_url>
YouTube Proxy:
GET /api/youtube/index.js?url=<encoded_url>
YouTube Embed:
GET /api/youtube/embed/index.js?url=<encoded_url>

Usage Example

https://your-domain.vercel.app/api/index.js?url=https%3A//google.com

Features

  • 🔄 Full HTML content rewriting and URL proxying
  • 🛡️ CORS handling and security header removal
  • 🖼️ Binary content support (images, fonts, etc.)
  • 🔍 Google search integration with form handling
  • 📺 YouTube video proxy with iframe injection
  • 🐛 Eruda debug tool integration
  • ⚡ TypeScript for better performance and reliability
